Item 1.0: Tabelas do sistema:
   > "unidade" : é a unidade fatec
       colunas: codigoUnidade (int) (PK); nomeFatecUnidade (Char 200); fatecDenominacaoOficial (CHar 200); nomeDiretor (Char 100)
   > "parametros" parametros do processamento
      colunas: ID (PK); anoReferencia (Int); mesReferencia(Int); matricula (FK - Tabela "usuario")
   > "usuario": é o usuario responsavel pelo sistema
       colunas:matricula (FK);codigoUnidade (FK - tabela Unidade); nomeFuncionario (Char 200); senhaLogin(password); indicadorAtivo (Boolean)    
   > "professor" contem a lista de professor
         colunas: matricula (PK); nomeProfessor(char:100); statusSituacao (int) (1-"Ativo", 9-"Suspenso de Processamento")
	               regimeJuridico(Char:20); cargaHoraria(Char:20);horaAtividade(int); 
	               HAE-O (Char:20); HAE-C (Char:20); obsManha (char:100);obsTarde (char:100);obsNoite (char:100);
   > "calendarioFeriados" contem feriados do ano
         colunas: ID(PK); descricao (char 100); dataFeriado (Date)
   > "calendarioAulasRemotas" são os intervalos, com a previsão de aulas remotas 
         colunas ID(PK); ID da tabela "curso" (FK); dataInicio (date); dataFim (date)
   > "disicplinaBasica": são as disciplinas básicas do instituicao
      colunas ID(PK); nomeDisicplina (char:100); cargaHoraria (int) 
   > "curso" São cursos previstos na instituição 
       colunas ID(PK); ID da tabela "curso" (FK); ID da tabela "Disciplina" (FK) 
   > "cursoDiscplina": são as disciplinas que fazem parte de um curso
      colunas ID(PK); ID da tabela "curso" (FK); ID da tabela "Disciplina" (FK)
 
  > "DisciplinaGrade": É a grade do curso com as discplinas
      colunas ID(PK); ID da tabela "cursoDisciplina" (FK); Periodo (char: 10 - manha/tarde/noite); diaSemana(Char:10- "Segunda";"Terca";"Quarta";""Quinta";"Sexta"; "Sabado") 
       
   > "gradeProfessor" são os dias de aulas do professor (grade horaria)
       colunas: ID(PK); matricula (Tabela "professor" - PK); ID da tabela "DisciplinaGrade" (FK)
